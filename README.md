# Endless Space Runner (Swipe Game)

## 🎮 Conceito
Jogo mobile estilo endless runner espacial, onde o jogador controla uma nave utilizando gestos de swipe para desviar de obstáculos e sobreviver o maior tempo possível.

## 📱 Controles (Swipe)
- Swipe para esquerda → move para esquerda
- Swipe para direita → move para direita
- Swipe para cima → move para cima
- Swipe para baixo → move para baixo

## 🎯 Objetivo
Sobreviver o maior tempo possível desviando dos obstáculos.

## ⚙️ Mecânicas
- Movimento por swipe em 4 direções
- Sistema de geração infinita de cenário (tiles)
- Sistema de colisão com Game Over
- Contador de tempo em tempo real

## 🖥️ Tecnologias
- Unity (Mobile)
- C#

## ▶️ Como jogar
Abra a cena principal e execute o jogo. Utilize swipes na tela para controlar o personagem.
